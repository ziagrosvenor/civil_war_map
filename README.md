DSA Civil War Assignment
========================

Getting started
---------------
Run an npm install.

Run gulp.

The resource location "http://localhost:9000/dsa-civilwar/?json=battles" in global.js
may need changing.

See store-data.php - used to store the battles data in a ready made SQL table.

Node modules
------------
- browser-sync
- cheerio
- gulp
- gulp-autoprefixer
- gulp-watch
- request
- underscore
